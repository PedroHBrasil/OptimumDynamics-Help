This will explain how to use and modify the help file if there are edits that need to be made

User interface is connected to the _theme file.  
The base.html holds the structure for all of the interface.
Style elements are all contained within the css folder and the base.css file
Each style piece is connected with an appropriate label,
If there is doubt as to what an element is, the easiest way to check it is to test the element and find the change locally

The theme is connected to the overall code through the mkdocs.yml file.
This also organizes the content for the table of contents.
The content of the help file is contained within the markdown files.
The .md files can be edited very similarly to a normal text file.
